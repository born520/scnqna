Selectum Noa Cam Ranh
Q - 객실 키 또는 팔찌 분실 했습니다. 재발급 받으려면 어떻게 해야 하나요 ?
A - 객실 키 / 1개 / 100,000₫
      팔찌 / 1개 / 1,000,000₫ ( 분실이 아닌 단순 훼손의 경우, 훼손된 팔찌 지참 후 리셉션 방문 시 무료 재발급 가능 )
Q - 체크인 전 / 후 호텔 내 부대 시설 이용이 가능할까요 ?
A - 워터파크 / 수영장 / 키즈 클럽 등의 부대 시설 모두 이용이 가능합니다.
Q - 셀렉토 키즈 클럽 운영 시간 및 베이티 시터 서비스 요청이 가능한가요 ?
A - 운영 시간 08:30 - 11:30 / 13:30 - 17:00 / 19:00 - 21:30
     베이비 시터 상주 08:30 - 11:30 / 13:30 - 17:00
Q - 셀렉토 키즈 클럽 베이티 시터 상주 시간 외에도 서비스 요청이 가능한가요 ?
A - 시간 외 서비스 이용을 원하실 경우, 원하는 시간 최소 4시간 전까지 리셉션 방문하신 후 예약 시 이용 가능합니다.
     비용 : 1H / 1 PAX / 300,000₫
Q - 스파 센터 운영 시간과 현장에서 예약 시 1인 비용은 어떻게 되나요 ?
A - 셀렉텀 노아 내 스파샵인 PURE SPA는 메인 빌딩의 3F에 있습니다. ( 운영 시간 / 10:00 - 18:00 )
     * 17:00 이후 예약 시, 1 PAX / 200,000₫의 추가 비용 발생
     현장 결제 시 1 PAX / 60Min / 1,198,000₫ 으로 카드 결제 또는 $ / ₫ 모두 사용 가능합니다.
Q - 타 호텔 투숙객도 이용료 지불 후 셀렉텀 노아 내 수영장과 워터 파크 이용할 수 있을까요 ?
A - 셀렉텀 노아 내 부대 시설은 투숙객에만 오픈 되는 시설로 타 호텔 투숙객은 이용 하실 수 없습니다.
Q - 파크 골프는 어떻게 이용할 수 있나요 ?
A - 현장에서 현금으로 결제 후 이용 가능합니다. 이용료와 장비 렌탈을 포함해 1 DAY / 1 PAX / 600,000₫ 입니다.
Q - 폼파티 일정은 어떻게 되나요 ?
A - 화 + 목 + 일 >> 15:30 MAIN POOL
     수 + 토 >> 21:00 S-ARENA
Q - 쿠킹 클래스는 무료로 이용 가능한가요 ?
A - 화 ( 반미 ) + 금 ( 스프링롤 ) + 일 ( 분짜 ) 호텔에서 쿠킹 클래스를 투숙객에게 무료 운영 중입니다.
