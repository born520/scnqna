Selectum Noa Cam Ranh
Q - 결혼 기념일로 방문 하는 고객님이 계십니다. 요청 가능한 서비스가 있을까요 ?
A - 무료 결혼 기념일 서비스 요청이 가능합니다. 객실 내 타올 장식과 장미꽃 데코레이션 서비스가 제공 됩니다. ( 증빙 자료 불필요 )
Q - 투숙 기간 내 생일자가 있습니다. 요청 가능한 서비스가 있을까요 ?
A - 무료 생일 서비스 요청이 가능합니다. 객실 내 타올 장식과 장미꽃 데코레이션 서비스, 작은 케이크가 제공 됩니다. ( 사전 요청 필수 / 체크아웃 생일자 요청 불가 )
Q - 커넥팅 룸 요청이 가능한가요 ?
A - 프리미어 디럭스 / 노아 가든 객실 한정 요청이 가능합니다. 1 TWN + 1 DBL 시 요청 가능하며 게런티 불가합니다.
Q - 성인 4인이 투숙할 수 있는 객실이 있을까요 ?
A - 객실 당 최대 성인 3인까지 투숙 가능으로 성인 4인일 경우 객실은 2개로 예약 하셔야 합니다.
Q - 기준 투숙 인원과 최대 투숙 인원은 어떻게 되나요 ?
A - 기준 투숙 성인 2인 / 최대 투숙 성인 3인 + 아동 1인 ( 성인 엑스트라 베드 필수 ) 입니다.
Q - 성인 2 + 아동 2 + 48개월 미만 1 투숙이 불가할까요 ?
A - 호텔 규정 상 원칙적으로는 투숙이 불가 하나, 호텔의 사전 컨펌 시 아동 엑스트라 베드 1개 추가 후 투숙이 가능합니다. ( 수배 전 담당자 확인 필수 )
Q - 룸 타입 변경 / 명단 / 베드 타입 / 미샌딩 / 식사 플랜 등 변경 요청할 팀이 있습니다. 체크인 며칠 전까지 요청 가능한가요 ?
A - 모든 변경은 체크인 7일 전까지 담당자에게 요청 가능합니다.
Q - 컷오프가 지난 예약건의 취소 또는 날짜 변경이 가능한가요 ?
A - 질병 등으로 인한 명확한 사유가 있는 취소 또는 날짜 변경은 영문 증빙 서류와 함께 요청이 가능합니다. 별도의 사유 및 증빙 서류가 없을 경우 모든 요청은 불가합니다.
